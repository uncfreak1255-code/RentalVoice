# Landing Page Rules (landing/)

## Stack
- Static HTML/CSS/JS — no framework
- Files: `index.html`, `styles.css`, `script.js`

## Constraints
- Must be lightweight and fast-loading (Core Web Vitals targets)
- Mobile-responsive design
- Privacy policy link at rentalvoice.app/privacy
- No external JS frameworks — vanilla only
- SEO-friendly: proper meta tags, structured data, semantic HTML
