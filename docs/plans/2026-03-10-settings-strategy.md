# Settings Screen Strategy — Competitive Moat

**Date:** 2026-03-10
**Status:** Draft / Not yet approved for implementation

## Core Thesis

Hostaway, Guesty, Enso will all add generic AI replies soon. Rental Voice's differentiator is voice accuracy — the AI sounds like the specific host. The Settings screen should make that moat visible and measurable.

## Current State (8 sections)

1. Connection (PMS status, manage connection)
2. Founder (founder access — dev-only)
3. AI Usage (drafts today, this month)
4. Memory Plan (commercial entitlements — jargon-heavy)
5. Auto-Pilot (toggle + confidence slider)
6. AI Learning (cultural tone, messages trained, profile strength, training history, knowledge base)
7. Notifications (push toggle, priority alerts)
8. Performance & Insights (approval rate, messages handled — only 2 stats)
9. About (version, privacy)

## Proposed Changes

### Priority 1: Voice Quality Dashboard (replace Performance & Insights)

- "Your Voice Match" percentage (reframed approval rate)
- Before/After draft comparison (week 1 vs. current)
- Learning velocity ("AI improved 12% this week")
- Guest satisfaction signal from positive post-AI replies

### Priority 2: Response Time Tracker

- Average response time vs. industry average
- Connection to Airbnb/VRBO ranking impact
- "AI is doing this for you" framing

### Priority 3: Section Reorder

1. **Your Voice** (AI Learning + Voice Quality merged — hero section)
2. **Auto-Pilot** (power feature)
3. **Usage & Plan** (metering, billing, memory status)
4. **Notifications**
5. **Connection** (setup — touched once)
6. **About**

### Priority 4: Remove/Hide

- Founder section → hidden behind 5-tap on version number
- Memory Plan → collapse into Usage ("Memory: Active/Limited")
- Cultural Tone toggle → auto-detect, remove toggle

### Not Yet

- Quick Replies/templates (table stakes, not differentiator)
- Guest sentiment dashboard (complex, better as standalone feature)

## Sales Pitch This Enables

"Rental Voice learns how YOU talk to guests. In 2 weeks it matches your voice 85%+, replies in under 5 minutes, and your Airbnb response rating goes to Superhost level."
